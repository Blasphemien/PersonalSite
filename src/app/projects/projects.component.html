John Brooks Engineering

Java, MySQL

Third year internship Java application to automate the process for selecting the correct sized couplings to connect a gear box to a specific conveyor belt motor based on factors such as torque (the amount of force needed to move an object), speed, friction and weight. Data for this application was stored on a local MySQL database which connected to the Java client which was made to be used cross-platform.

The design analysis and techniques of the project included UML using Visual Paradigm to layout the DB entities before they were put into development, the Software Development Life Cycle methodology, unit testing in jUnit, Agile project management using SCRUM to manage development phases as well as documentation which had to be signed off by the client.

  • Static table on GUI to guide the user of the boundaries between the different ratios of tourqe, speed and friction of a motor and gearbox
  • Calculations are stored in a centralized MySQL DB
  • User has the option to make print outs of past calculations for review

StarWarsCharacterNames (npm pkg & JS Library)

   
Javascript

starwars-character-names is a JS library published to npm. Its purpose is to provide the user an easy API for getting random names of starwars characters

Build tools
  • Testing: mocha, chai
  • Release: semantic-release & commitizen
  • Build: Webpack
  • Intergration: Travis CI
Install

npm install --save starwars-character-names
Usage

// require module
var starWarsNames = require('starwars-character-names');

// returns an array of all names
var allNames = starWarsNames.filter.all;

// returns single random name
var randomName = starWarsNames.filter.random();

// returns an array of n number of names
var fourNames = starWarsNames.filter.random(number);

// returns array of names with the first name beginning with user defined char
var firstNamesByChar = starWarsNames.filter.byFirstCharOfFirstName(char);

// returns array of names with the second name beginning with user defined char
var secondNamesByChar = starWarsNames.filter.byFirstCharOfSecondName(char);
